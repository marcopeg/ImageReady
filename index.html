ImageReady jQuery Plugin

Check an image or a list of images to be loaded and ready to be scripted

What does this plugin do?

ImageReady check a list of images to be ready to be handled by jQuery.
An image must to be complete and expose the naturalWidth attribute to be considered ready!

How do I use this demo?

Configuration:

mode
[each|all] (default "each")
step
(default 50) delay between checking cycle steps. Lower the value, lower the performances, higher the value lower the precision!
timeout
(default 5000) it is the maximum time allowed to an image to download.
You can setup this value based on an estimate download time.
stackTimeout
(default "auto") it is the maximum time allowed to a group of images to download.
If you use auto the script will calculate a value based on the timeout property increasing it based on images count.
checkingClass
(default: "imageready-checking") a CSS class assigned to the image DOM while checking.
successClass
(default: "imageready-success") a CSS class assigned to the image DOM when success.
errorClass
(default: "imageready-error") a CSS class assigned to the image DOM when timeout happen.
success(cfg)
callback throwed when an image (or a group of images) is ready.
It receive the image itself as context in "each" mode.
It receive the images stack as context in "all" mode.
onError(cfg)
callback throwed when an image (or a group of images) fall into timeout.
It receive the image itself as context in "each" mode.
It receive the images stack as context in "all" mode.
stepSuccess(cfg,stack)
It works like success() but is used internally by the "all" mode.
It is throwed on each single image in a stack.
stepOnError(cfg,stack)
It works like onError() but is used internally by the "all" mode.
It is throwed on each single image in a stack.
